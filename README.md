## 如何添加到桌面（推荐）

**Chrome / Edge**  
1. 打开网站  
2. 点击地址栏左侧小锁 → 安装应用 / 创建快捷方式  
3. 选择「创建桌面图标」或「固定到任务栏」

**手机端**  
Safari → 分享 → 添加到主屏幕  
Chrome → 菜单 → 添加到主屏幕

## 常见问题

**Q：为什么有时候打不开？**  
A：域名可能被运营商劫持或DNS污染，请尝试备用地址或更换DNS（推荐1.1.1.1 / 8.8.8.8）

**Q：可以投稿网站吗？**  
A：欢迎！请提交issue，格式参考：  
站点名称 | 一句话描述 | 网址 | 推荐分类

**Q：你们会收集我的浏览记录吗？**  
A：不会。本站不使用任何第三方统计、分析工具，所有数据仅保存在你本地。
